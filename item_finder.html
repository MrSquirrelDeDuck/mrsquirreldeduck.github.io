Instructions

In the text box type the name of your item.

It is case sensitive and must match perfectly.

For example, "Dirt Block" is correct so "Dirt" will not work.

Because of this, if you put in a block name and nothing pops up you may need to add "block" to the end



Once you type in the name of the item to lookup, press the "Find!" button.


When you click "Find!" if what you put into the text box is a valid item chests will light up

in the chest indicator. If a chest has an "x" on it that means that item goes in that chest.
