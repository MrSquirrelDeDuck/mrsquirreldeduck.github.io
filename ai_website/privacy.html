School Privacy

Introduction

This privacy policy describes how the student data collected by the One Page School Application is used, and how it is protected.

What Data Is Collected

The following data is collected:

  • Name, email address, and a password (used to login) of the administrator
  • Names of students
  • Names of parents (optional)
  • School name, address, and phone number
  • School year
  • Classes
  • Attendance
  • Grades

How Data Is Used

All of the data is stored in a database and is used to display the information on the website when you are logged in. The email address and password are used to log you in to the admin panel.

How Data Is Protected

The email address and password of the administrator are encrypted with a salted hash. The salted hash means that it would be very difficult to crack the hash and figure out the original password. The administrator's email address is also used as an identifier in the database, so it is not possible to log in using an email address that is not in the database.

The database is stored in a directory on the server that is not accessible to the public. That means that nobody can access the database except for the website itself.
